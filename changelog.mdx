## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

- Added real-time collaborative editing for documentation pages
- Implemented advanced search with filters and sorting options
- Introduced dark mode support across all documentation interfaces

- Enhanced API response times by 40% through optimized database queries
- Improved webhook delivery reliability with automatic retry mechanisms
- Updated user interface components for better accessibility compliance

<Update label="2024-09-20" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Complete API redesign with RESTful endpoints for better integration
- Added support for multiple repository connections per documentation space
- Implemented granular permission controls for team collaboration

## Breaking Changes

- API endpoints now require authentication headers for all requests
- Webhook payload structure has been updated with new event types
- Configuration file format changed from JSON to YAML for better readability

</Update>
